## 🖥️ System Overview

The portfolio is built with a focus on "Engineering Intelligence," bridging the gap between low-level logic and high-level software architecture. It features a custom-built Bento-grid layout, real-time system monitors, and a dynamic charting engine.

### Core Features
- **Dual-Mode Interface:** Seamless transition between `Light-Theme` (High-Contrast Emerald) and `Dark-Theme` (Neon Cyan-Green).
- **Dynamic Skill Distribution:** An SVG-based circular charting system that visualizes technical proficiency.
- **Bento Grid Projects:** A modern layout for showcasing primary works and a "System Archive" side-drawer for legacy projects.
- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop environments.
- **Performance Focused:** Minimal dependencies, using native CSS variables and vanilla JavaScript for animations and state management.

## 🛠️ Technical Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Grid, Flexbox), Vanilla JavaScript (ES6+).
- **Animation:** CSS Keyframes and transition logic.
- **Icons:** FontAwesome 6.5.0.
- **Typography:** Inter (Sans-serif) & JetBrains Mono (System/Monospace).
